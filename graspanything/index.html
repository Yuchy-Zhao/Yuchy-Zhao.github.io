GraspAnything: Interactive General Object Grasping

Internship Program

Overview

General object grasping is a fundamental problem in robotics and has many applications in industry. In this project, we designed an interactive, general grasping framework. Users can complete the task of grasping objects with a robotic arm by simply clicking on the target object in the interactive interface. The system pipeline is as follows: interactive segmentation is achieved through the SAM model with the input color and user prompt. Using the generated mask and the input depth, the 3D point cloud of the target object is obtained. Candidate grippers are then generated using the general grasping network GraspNet, and finally, the optimal gripper is selected to grasp the target.

We selected six different objects to verify the effectiveness of the general object grasping. From top to bottom are the input image, interactive segmentation results, candidate grippers, and the optimal gripper. The results show that the system can accurately segment the target object and generate a reasonable gripper.
